# ft containers
This is a project from 42 Network Cursus.
>Because the STL containers in the standard library are not good, I decided to reimplement them from scratch.

## Usage

```c++
#include "map/Map.hpp"
#include "vector/vector.hpp"
#include "stack/stack.hpp"


int main()
{
  ft::vector hego;
  ft::stack  overflow;
  ft::map    google;
}
```
